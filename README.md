# 🤖 AI Empowered Virtual Mouse

An AI-powered virtual mouse system that allows you to control your computer cursor using a webcam or other input devices.

---

## 📁 Project Structure

```
.
├── collect_data.py       # Collect data using webcam or other input
├── train_model.py        # Train model using collected data
├── cursor_controller.py  # Main mouse controller application
```

---

## 📌 Description

### 1. collect_data.py
Collects data using a webcam or other input devices.  
This data is used to build a dataset for training the model.

### 2. train_model.py
Trains a machine learning model using the collected dataset.  
Better data = better performance.

### 3. cursor_controller.py
Main application that uses the trained model to control the mouse cursor in real-time.

---

## ⚙️ How It Works

1. Collect data using your webcam
2. Train the model with collected data
3. Run the controller to move the cursor using AI

---

## 💡 Tips for Better Accuracy

- Train the model in **different lighting conditions**
- Use **varied backgrounds**
- Capture **multiple hand positions and angles**
- Increase dataset size for better performance
